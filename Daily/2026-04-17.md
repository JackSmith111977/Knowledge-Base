**认证体系重点（面试最高频）：**
- OAuth 2.1 四种授权模式：authorization code / implicit / client credentials / resource owner password
- PKCE 流程：为什么 PKCE 取代了 implicit？code_challenge / code_verifier 生成
- JWT 三大部分：Header.Payload.Signature，HS256 vs RS256 区别
- JWT 常见漏洞：alg=none 攻击、弱密钥爆破、过期时间设置
- Cookie 安全属性：SameSite（Strict/Lax/None）、Secure、HttpOnly、Domain/Path
- Cookie 分区机制：CHIPS、Third-Party Cookie 淘汰时间线
- XSS 窃取 Token 的完整攻击链 + 防护方案
