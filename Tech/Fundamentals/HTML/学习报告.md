## 错题记录

### 错题 1：HTML5 废除的标签

- **章节**：第 1 章 HTML 概述
- **问题**：HTML5 相比 HTML4 新增了哪些核心能力？以下说法哪个是错误的？
- **用户回答**：D（误以为 HTML5 废除了 LocalStorage API）
- **正确答案**：C（HTML5 **废除**了 `<font>` 和 `<center>` 标签以保持向后兼容 — 这是错误的说法）
- **纠正**：HTML5 移除了 `<font>`、`<center>`、`<frame>` 等过时标签，LocalStorage 是 HTML5 **新增**的 API

### 错题 2：Canvas vs SVG 选型

- **章节**：第 7 章 图形绘制
- **问题**：数据可视化仪表盘（静态图表 + 点击交互），应选哪种方案？
- **用户回答**：D（用 `<canvas>` 加 `<div>` 覆盖层）
- **正确答案**：B（SVG，因为图表是静态的且需要交互和 DOM 事件）
- **纠正**：Canvas 需要手动计算鼠标坐标判断点击，SVG 原生支持 DOM 事件。数据可视化图表应优先 SVG

## 核心要点总结

### 已牢固掌握的知识点
- HTML 文档结构骨架（DOCTYPE → html → head → body）
- 语义化标签体系及其三大价值（SEO/无障碍/可读性）
- 文本格式化语义（strong vs b, em vs i）
- 超链接安全（target="_blank" + rel="noopener"）
- 图片无障碍（alt="" 用于装饰性图片）
- 表格语义结构（caption/thead/tbody/tfoot）
- 表单基础与 input 类型
- 表单原生验证 API
- datalist vs select 区别
- 表单无障碍（label 必关联）
- 音视频标签及自动播放限制
- 媒体 API（play/pause/currentTime）
- Web Components 三大技术（Custom Elements/Shadow DOM/Templates）
- 自定义元素生命周期
- 基础元数据（charset/viewport/description/robots）
- Open Graph 与结构化数据（JSON-LD）
- Canonical URL 与 favicon 配置
