## 1. 概述

### 1.1 什么是滑动窗口

**滑动窗口 (Sliding Window)** 是一种算法技巧，适用于处理连续子数组 (或子串) 的问题。

**核心思想：** 用两个指针 (left 和 right) 维护一个"窗口"，在原数组或字符串上滑动这个窗口来寻找满足条件的区间。

**形象理解：**

```
想象你站在一扇窗户前看风景：
- 窗户可以左右移动 (滑动)
- 窗户可以变大变小 (伸缩)
- 你通过窗户观察外面的景色 (寻找最优解)
```

### 1.2 滑动窗口的本质

滑动窗口本质是**同向双指针**的一种应用：

| 指针 | 作用 | 移动方向 |
|------|------|----------|
| **左指针 (left)** | 窗口左边界，负责收缩窗口 | 向右移动 |
| **右指针 (right)** | 窗口右边界，负责扩大窗口 | 向右移动 |

**窗口表示：** `[left, right)` 或 `[left, right]`（左闭右开或左闭右闭）

### 1.3 为什么需要滑动窗口

**问题场景：** 求满足条件的最长/最短子数组、子字符串。

**暴力解法：**
```javascript
// 枚举所有子数组，时间复杂度 O(n²)
for (let i = 0; i < n; i++) {
  for (let j = i; j < n; j++) {
    // 检查 [i, j] 是否满足条件
  }
}
```

**滑动窗口优化：**
```javascript
// 双指针滑动，时间复杂度 O(n)
let left = 0, right = 0;
while (right < n) {
  // 扩大窗口
  // 满足条件时收缩窗口
  right++;
}
```

**性能对比：**

| 方法 | 时间复杂度 | n=10⁵ 时运算次数 |
|------|-----------|-----------------|
| 暴力法 | O(n²) | 10¹⁰ (超时) |
| 滑动窗口 | O(n) | 10⁵ (通过) |

### 1.4 适用场景

滑动窗口适用于以下典型问题：

| 问题类型 | 描述 | 典型题目 |
|----------|------|----------|
| **固定长度窗口** | 窗口大小固定为 k | 长度为 k 的子数组最大和 |
| **最长子串/子数组** | 找满足条件的最长区间 | 无重复字符的最长子串 |
| **最小子串/子数组** | 找满足条件的最小区间 | 最小覆盖子串 |
| **窗口内统计** | 统计窗口内的特定信息 | 滑动窗口最大值 |

## 2. 核心思想

### 2.1 滑动窗口的本质

滑动窗口本质是**同向双指针**的一种应用：

| 指针 | 作用 | 移动方向 |
|------|------|----------|
| **左指针 (left)** | 窗口左边界，负责收缩窗口 | 向右移动 |
| **右指针 (right)** | 窗口右边界，负责扩大窗口 | 向右移动 |

**窗口表示：** `[left, right]`（左闭右闭，推荐与代码实现一致）

### 2.2 核心思路：三步法

灵茶山艾府将滑动窗口总结为简洁的**三步法**，适用于所有滑动窗口题目：

#### 定长滑动窗口：入 - 更新 - 出

```
窗口右端点在 i 时，由于窗口长度为 k，所以窗口左端点为 i - k + 1

第一步：入   → 下标为 i 的元素进入窗口，更新相关统计量
第二步：更新 → 更新答案（最大值/最小值）
第三步：出   → 下标为 i - k + 1 的元素离开窗口，更新统计量
```

#### 不定长滑动窗口：进窗口 - 出窗口 - 更新答案

```
第一步：进窗口 → right 指针向右移动，将新元素加入窗口
第二步：出窗口 → 如果窗口不满足条件，left 指针持续右移，将元素移出窗口
第三步：更新答案 → 在窗口状态改变后，更新最终结果
```

### 2.3 为什么需要滑动窗口

**问题场景：** 求满足条件的最长/最短子数组、子字符串。

**暴力解法：**
```javascript
// 枚举所有子数组，时间复杂度 O(n²)

    // 检查 [i, j] 是否满足条件

```

**滑动窗口优化：**

## 3. 定长滑动窗口模板

### 3.1 核心模板（灵茶山艾府风格）

**适用场景：** 窗口大小固定为 k 的问题。

**核心思路：入 - 更新 - 出**

```javascript
/**
 * @param {number[]} nums - 输入数组
 * @param {number} k - 窗口大小
 * @return {number} - 结果
 */
function fixedSlidingWindow(nums, k) {
  const n = nums.length;
  let sum = 0;      // 窗口状态（示例为求和，实际根据题目变化）
  let ans = 0;      // 答案

  // 枚举右端点
  for (let i = 0; i < n; i++) {
    // 第一步：入 - 下标为 i 的元素进入窗口
    sum += nums[i];

    // 如果窗口尚未形成（前 k-1 个元素），继续循环
    if (i < k - 1) {
      continue;
    }

    // 第二步：更新 - 更新答案
    ans = Math.max(ans, sum);

    // 第三步：出 - 下标为 i - k + 1 的元素离开窗口
    sum -= nums[i - k + 1];
  }

  return ans;
}
```

**关键点注释：**
```
窗口形成过程（k = 3 的例子）：

i = 0:  [0]              → i < k-1，继续
i = 1:  [0, 1]           → i < k-1，继续
i = 2:  [0, 1, 2]        → 第一个完整窗口，更新答案，0 出窗
i = 3:  [1, 2, 3]        → 第二个窗口，更新答案，1 出窗
i = 4:  [2, 3, 4]        → 第三个窗口，更新答案，2 出窗
                        ...
```

### 3.2 两种写法对比

**写法一：循环外单独处理第一个窗口**

```javascript
// 初始化第一个窗口
for (let i = 0; i < k; i++) {
  sum += nums[i];
}
ans = sum;

// 滑动窗口
for (let i = k; i < n; i++) {
  sum += nums[i];           // 入
  ans = Math.max(ans, sum); // 更新
  sum -= nums[i - k + 1];   // 出（注意这里 i 已经是新窗口的右端点）
}
```

**写法二：循环内统一处理（推荐）**

```javascript
for (let i = 0; i < n; i++) {
  sum += nums[i];                    // 入
  if (i < k - 1) continue;           // 窗口未形成
  ans = Math.max(ans, sum);          // 更新
  sum -= nums[i - k + 1];            // 出
}
```

**两种写法等价，写法二代码更简洁。**

### 3.3 窗口左端点公式推导

**问题：** 为什么窗口右端点为 `i` 时，左端点是 `i - k + 1`？

**推导：**
```
对于闭区间 [L, R]，区间内元素个数 = R - L + 1

已知：窗口大小 = k，右端点 R = i
求：左端点 L

k = R - L + 1
k = i - L + 1
L = i - k + 1
```

**验证（k = 3）：**
- i = 2 时，L = 2 - 3 + 1 = 0，窗口 [0, 2]，共 3 个元素 ✓
- i = 3 时，L = 3 - 3 + 1 = 1，窗口 [1, 3]，共 3 个元素 ✓

## 4. 不定长滑动窗口模板

### 4.1 核心模板（灵茶山艾府风格）

**适用场景：** 窗口大小动态变化，根据条件决定扩大或缩小。

**核心思路：进窗口 - 出窗口 - 更新答案**

```javascript
/**
 * @param {number[]} nums - 输入数组
 * @param {number} target - 目标和
 * @return {number} - 最短子数组长度
 */
function variableSlidingWindow(nums, target) {
  let left = 0;
  let sum = 0;           // 窗口状态
  let ans = Infinity;    // 答案

  // 枚举右端点
  for (let right = 0; right < nums.length; right++) {
    // 第一步：进窗口 - 将 right 指向的元素加入窗口
    sum += nums[right];

    // 第二步：出窗口 - 如果窗口不满足条件，left 指针持续右移
    // 注意：这里用 while 而不是 if，因为可能需要多次收缩
    while (sum >= target) {
      sum -= nums[left];
      left++;
    }

    // 第三步：更新答案 - 在窗口状态改变后更新
    // 注意：根据题目要求，更新位置可能在 while 之前或之后
    ans = Math.min(ans, right - left + 1);
  }

  return ans === Infinity ? 0 : ans;
}
```

### 4.2 两类不定长窗口对比

根据题目要求，分为两种类型：

| 类型 | 目标 | 收缩条件 | 更新时机 | 典型题目 |
|------|------|----------|----------|----------|
| **求最长/最大** | `max(ans, len)` | 窗口**不合法**时收缩 | 窗口**合法**时更新 | 无重复字符的最长子串 |
| **求最短/最小** | `min(ans, len)` | 窗口**合法**时收缩 | 窗口**合法**时更新 | 长度最小的子数组 |

**求最长模板：**
```javascript
let left = 0, ans = 0;
const windowState = new Set();

for (let right = 0; right < n; right++) {
  // 进窗口
  windowState.add(nums[right]);

  // 出窗口：窗口不合法时收缩
  while (窗口不合法) {
    windowState.delete(nums[left]);
    left++;
  }

  // 更新答案：窗口合法时更新
  ans = Math.max(ans, right - left + 1);
}
```

**求最短模板：**
```javascript
let left = 0, ans = Infinity;
let sum = 0;

for (let right = 0; right < n; right++) {
  // 进窗口
  sum += nums[right];

  // 出窗口：窗口合法时收缩
  while (窗口合法) {
    ans = Math.min(ans, right - left + 1);
    sum -= nums[left];
    left++;
  }
}
```

### 4.3 子数组个数问题

不定长滑动窗口还可以用于**统计满足条件的子数组个数**。

**类型一：越短越合法（一般写 `ans += right - left + 1`）**

```javascript
// 典型题目：713. 乘积小于 K 的子数组
let left = 0, ans = 0;
let product = 1;

for (let right = 0; right < n; right++) {
  product *= nums[right];

  // 窗口不合法时收缩
  while (product >= k && left <= right) {
    product /= nums[left];
    left++;
  }

  // 以 right 结尾的合法子数组个数 = right - left + 1
  ans += right - left + 1;
}
```

**类型二：越长越合法（一般写 `ans += left`）**

```javascript
// 典型题目：1358. 包含所有三种字符的子字符串数目
let left = 0, ans = 0;
const cnt = [0, 0, 0];

for (let right = 0; right < n; right++) {
  cnt[s[right] - 'a']++;

  // 窗口合法时，记录答案并收缩
  while (窗口合法) {
    cnt[s[left] - 'a']--;
    left++;
  }

  // 以 right 结尾的合法子数组个数 = left
  ans += left;
}
```

**类型三：恰好型（需要维护两个指针）**

```javascript
// 典型题目：930. 和相同的二元子数组
// 恰好 = 最多 (goal) - 最多 (goal - 1)
function atMost(nums, goal) {
  let left = 0, sum = 0, ans = 0;
  for (let right = 0; right < nums.length; right++) {
    sum += nums[right];
    while (sum > goal && left <= right) {
      sum -= nums[left];
      left++;
    }
    ans += right - left + 1;
  }
  return ans;
}

function exactly(nums, goal) {
  return atMost(nums, goal) - atMost(nums, goal - 1);
}
```

## 5. 定长窗口题型

### 5.1 基础题（入门必做）

**1456. 定长子串中元音的最大数目**（裸题）

```javascript
/**
 * @param {string} s
 * @param {number} k
 * @return {number}
 */
function maxVowels(s, k) {
  const vowels = new Set(['a', 'e', 'i', 'o', 'u']);
  let cnt = 0;
  let ans = 0;

  for (let i = 0; i < s.length; i++) {
    // 入：当前字符进入窗口
    if (vowels.has(s[i])) {
      cnt++;
    }

    // 窗口尚未形成
    if (i < k - 1) {
      continue;
    }

    // 更新：更新最大元音数
    ans = Math.max(ans, cnt);

    // 出：窗口左端点字符离开窗口
    if (vowels.has(s[i - k + 1])) {
      cnt--;
    }
  }

  return ans;
}
```

**关键注释：**
- `i < k - 1`：前 k-1 个字符用于初始化窗口，不更新答案
- `i - k + 1`：窗口左端点，该位置的字符即将离开窗口

**643. 子数组最大平均数 I**（裸题）

```javascript
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {number}
 */
function findMaxAverage(nums, k) {
  let sum = 0;
  let ans = -Infinity;

  for (let i = 0; i < nums.length; i++) {
    // 入
    sum += nums[i];

    if (i < k - 1) continue;

    // 更新：第一次时初始化 ans
    ans = Math.max(ans, sum / k);

    // 出
    sum -= nums[i - k + 1];
  }

  return ans;
}
```

**2090. 半径为 k 的子数组平均值**

```javascript
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {number[]}
 */
function getAverages(nums, k) {
  const n = nums.length;
  const ans = new Array(n).fill(-1);
  let sum = 0;
  const windowSize = 2 * k + 1;  // 窗口长度为 2k+1

  for (let i = 0; i < n; i++) {
    // 入
    sum += nums[i];

    if (i < windowSize - 1) continue;

    // 更新：中心点下标为 i - k
    ans[i - k] = Math.floor(sum / windowSize);

    // 出：nums[i - 2*k] 离开窗口
    sum -= nums[i - 2 * k];
  }

  return ans;
}
```

**关键注释：**
- 窗口大小为 `2k + 1`（半径为 k，左右各 k 个元素）
- 中心点下标为 `i - k`
- 只有当中心点左右都有 k 个元素时，平均值才有效

**2379. 得到 K 个黑块的最少涂色次数**

```javascript
/**
 * @param {string} blocks
 * @param {number} k
 * @return {number}
 */
function minimumRecolors(blocks, k) {
  let cnt = 0;  // 统计黑块数量
  let ans = k;  // 最坏情况需要涂 k 次

  for (let i = 0; i < blocks.length; i++) {
    // 入
    if (blocks[i] === 'B') {
      cnt++;
    }

    if (i < k - 1) continue;

    // 更新：需要涂色的次数 = k - 当前黑块数
    ans = Math.min(ans, k - cnt);

    // 出
    if (blocks[i - k + 1] === 'B') {
      cnt--;
    }
  }

  return ans;
}
```

**关键注释：**
- 问题转化：找长度为 k 的窗口中，白块最少的窗口
- 答案 = k - 窗口中黑块数量

**2841. 几乎唯一子数组的最大和**

```javascript
/**
 * @param {number[]} nums
 * @param {number} m
 * @param {number} k
 * @return {number}
 */
function maxSum(nums, m, k) {
  const cnt = new Map();  // 统计窗口内各元素出现次数
  let sum = 0;
  let ans = 0;
  let distinctCnt = 0;    // 不同元素个数

  for (let i = 0; i < nums.length; i++) {
    // 入
    sum += nums[i];
    const inCount = (cnt.get(nums[i]) || 0);
    if (inCount === 0) {
      distinctCnt++;
    }
    cnt.set(nums[i], inCount + 1);

    if (i < k - 1) continue;

    // 更新：只有不同元素个数 >= m 时才更新答案
    if (distinctCnt >= m) {
      ans = Math.max(ans, sum);
    }

    // 出
    const outNum = nums[i - k + 1];
    const outCount = cnt.get(outNum);
    if (outCount === 1) {
      distinctCnt--;
    }
    cnt.set(outNum, outCount - 1);
  }

  return ans;
}
```

**关键注释：**
- 需要同时维护窗口和 和 不同元素个数
- 出窗口时，如果某个元素计数降为 0，不同元素个数减 1

## 6. 不定长窗口题型

### 6.1 求最长/最大（窗口不合法时收缩）

**典型题目：** 无重复字符的最长子串

**解题思路：**
- 右指针扩展窗口
- 当窗口**不合法**（出现重复）时，左指针收缩
- 在窗口**合法时**更新最大长度

```javascript
// 通用模板
let left = 0, ans = 0;
const windowState = new Set();

for (let right = 0; right < n; right++) {
  // 进窗口
  windowState.add(nums[right]);

  // 出窗口：窗口不合法时收缩
  while (窗口不合法) {
    windowState.delete(nums[left]);
    left++;
  }

  // 更新答案：窗口合法时更新
  ans = Math.max(ans, right - left + 1);
}
```

### 6.2 求最短/最小（窗口合法时收缩）

**典型题目：** 长度最小的子数组、最小覆盖子串

**解题思路：**
- 右指针扩展窗口
- 当窗口**合法**时，左指针收缩
- 在窗口**合法时**更新最小长度

```javascript
// 通用模板
let left = 0, ans = Infinity;
let sum = 0;

for (let right = 0; right < n; right++) {
  // 进窗口
  sum += nums[right];

  // 出窗口：窗口合法时收缩
  while (窗口合法) {
    ans = Math.min(ans, right - left + 1);
    sum -= nums[left];
    left++;
  }
}
```

### 6.3 求子数组个数

**类型一：越短越合法（`ans += right - left + 1`）**

窗口越小越容易满足条件，统计以 right 结尾的合法子数组个数。

```javascript
// 典型题目：713. 乘积小于 K 的子数组
let left = 0, ans = 0;
let product = 1;

for (let right = 0; right < n; right++) {
  product *= nums[right];

  // 窗口不合法时收缩
  while (product >= k && left <= right) {
    product /= nums[left];
    left++;
  }

  // 以 right 结尾的合法子数组个数 = right - left + 1
  ans += right - left + 1;
}
```

**类型二：越长越合法（`ans += left`）**

窗口越大越容易满足条件。

```javascript
// 典型题目：1358. 包含所有三种字符的子字符串数目
let left = 0, ans = 0;
const cnt = [0, 0, 0];

for (let right = 0; right < n; right++) {
  cnt[s[right] - 'a']++;

  // 窗口合法时，记录答案并收缩
  while (窗口合法) {
    cnt[s[left] - 'a']--;
    left++;
  }

  // 以 right 结尾的合法子数组个数 = left
  ans += left;
}
```

**类型三：恰好型（最多 (goal) - 最多 (goal-1)）**

```javascript
// 典型题目：930. 和相同的二元子数组
// 恰好 = 最多 (goal) - 最多 (goal - 1)
function atMost(nums, goal) {
  let left = 0, sum = 0, ans = 0;
  for (let right = 0; right < nums.length; right++) {
    sum += nums[right];
    while (sum > goal && left <= right) {
      sum -= nums[left];
      left++;
    }
    ans += right - left + 1;
  }
  return ans;
}

function exactly(nums, goal) {
  return atMost(nums, goal) - atMost(nums, goal - 1);
}
```

### 6.1 LeetCode 3. 无重复字符的最长子串

**题目描述：**
给定一个字符串 s，返回其中不包含重复字符的最长子串的长度。

```
示例 1:
输入: s = "abcabcbb"
输出: 3
解释: 最长无重复子串是 "abc"

示例 2:
输入: s = "bbbbb"
输出: 1

示例 3:
输入: s = "abba"
输出: 2
```

**解题思路：**
1. 使用 Set 记录窗口内的字符
2. 右指针扩展窗口
3. 遇到重复字符时，左指针收缩直到无重复
4. 更新最大长度

**代码实现：**
```javascript
/**
 * @param {string} s
 * @return {number}
 */
function lengthOfLongestSubstring(s) {
  const windowSet = new Set();
  let left = 0, maxLength = 0;

  for (let right = 0; right < s.length; right++) {
    // 遇到重复字符，收缩左边界
    while (windowSet.has(s[right])) {
      windowSet.delete(s[left]);
      left++;
    }

    // 加入当前字符
    windowSet.add(s[right]);

    // 更新最大长度
    maxLength = Math.max(maxLength, right - left + 1);
  }

  return maxLength;
}
```

**复杂度分析：**
- 时间复杂度：O(n)，左右指针各遍历一次
- 空间复杂度：O(min(m, n))，m 为字符集大小

### 6.2 LeetCode 76. 最小覆盖子串

**题目描述：**
给定字符串 s 和 t，在 s 中找一个最短的子串，使它包含 t 中的全部字符。

```
示例:
输入: s = "ADOBECODEBANC", t = "ABC"
输出: "BANC"
```

**解题思路：**
1. 使用 Map 记录 t 中每个字符需要的数量
2. 右指针扩展窗口，统计窗口内字符
3. 当窗口满足条件时，尝试收缩左边界
4. 更新最小覆盖子串

**代码实现：**
```javascript
/**
 * @param {string} s
 * @param {string} t
 * @return {string}
 */
function minWindow(s, t) {
  const need = new Map();
  for (const char of t) {
    need.set(char, (need.get(char) || 0) + 1);
  }

  let needCount = t.length;
  let left = 0, right = 0;
  let minLength = Infinity;
  let result = '';

  while (right < s.length) {
    const rightChar = s[right];

    // 右边界字符加入窗口
    if (need.has(rightChar) && need.get(rightChar) > 0) {
      needCount--;
    }
    need.set(rightChar, (need.get(rightChar) || 0) - 1);

    // 窗口满足条件时，尝试收缩
    while (needCount === 0) {
      if (right - left + 1 < minLength) {
        minLength = right - left + 1;
        result = s.slice(left, right + 1);
      }

      const leftChar = s[left];
      need.set(leftChar, (need.get(leftChar) || 0) + 1);
      if (need.get(leftChar) > 0) {
        needCount++;
      }
      left++;
    }

    right++;
  }

  return result;
}
```

**关键点：**
- `need` 值可正可负：正数表示还缺多少，负数表示多余多少
- `needCount` 记录还需要的字符总数，为 0 时表示窗口满足条件

### 6.3 LeetCode 209. 长度最小的子数组

**题目描述：**
给定一个正整数数组 nums 和目标值 target，找出和 ≥ target 的最短连续子数组长度。

```
示例:
输入: target = 7, nums = [2,3,1,2,4,3]
输出: 2
解释: 子数组 [4,3] 是最短的和 ≥ 7 的子数组
```

**代码实现：**
```javascript
/**
 * @param {number} target
 * @param {number[]} nums
 * @return {number}
 */
function minSubArrayLen(target, nums) {
  let left = 0, sum = 0;
  let minLength = Infinity;

  for (let right = 0; right < nums.length; right++) {
    // 扩大窗口
    sum += nums[right];

    // 窗口满足条件时，尝试收缩
    while (sum >= target) {
      minLength = Math.min(minLength, right - left + 1);
      sum -= nums[left];
      left++;
    }
  }

  return minLength === Infinity ? 0 : minLength;
}
```

## 7. 易错点与技巧

### 7.1 常见错误

| 错误 | 说明 | 正确做法 |
|------|------|----------|
| **区间开闭混淆** | `[left, right)` vs `[left, right]` | 统一使用一种表示，推荐左闭右开 |
| **收缩条件错误** | 求最长/最短的收缩时机混淆 | 最长：不合法时收缩；最短：合法时收缩 |
| **更新时机错误** | 在错误的位置更新结果 | 合法窗口时更新（最长/最短都一样） |
| **忘记初始化** | 第一个窗口未处理 | 固定窗口先初始化第一个窗口 |

```javascript
  // 入
  sum += nums[i];

  if (i < k - 1) continue;  // 窗口尚未形成

  // 更新
  ans = Math.max(ans, sum);

  // 出
  sum -= nums[i - k + 1];
}
```

### 不定长滑动窗口（进窗口 - 出窗口 - 更新答案）

**求最长：**
```javascript
let left = 0, ans = 0;
const windowState = new Set();

for (let right = 0; right < n; right++) {
  // 进窗口
  windowState.add(nums[right]);

  // 出窗口：窗口不合法时收缩
  while (窗口不合法) {
    windowState.delete(nums[left]);
    left++;
  }

  // 更新答案
  ans = Math.max(ans, right - left + 1);
}
```

**求最短：**
```javascript
let left = 0, ans = Infinity;
let sum = 0;

for (let right = 0; right < n; right++) {
  // 进窗口
  sum += nums[right];

  // 出窗口：窗口合法时收缩
  while (窗口合法) {
    ans = Math.min(ans, right - left + 1);
    sum -= nums[left];
    left++;
  }
}
```

**求个数（越短越合法）：**
```javascript
let left = 0, ans = 0;
let product = 1;

for (let right = 0; right < n; right++) {
  product *= nums[right];

  while (product >= k && left <= right) {
    product /= nums[left];
    left++;
  }

  ans += right - left + 1;
}
```
