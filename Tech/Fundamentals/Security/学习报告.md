## 核心要点总结

### 1. OWASP Top 10 前三
访问控制失效（A01）、加密机制失效（A02）、注入攻击（A03）

### 2. XSS 防御三件套
输入过滤（DOMPurify 白名单） → 输出编码（textContent） → 安全 Cookie（HttpOnly + Secure + SameSite）

### 3. CSRF 防御核心
CSRF Token（随机不可预测） + SameSite Cookie（Lax/Strict）

### 4. 注入攻击防御原则
数据与命令分离 → 参数化查询、execFile 替代 exec、输入转义

### 5. 密码与会话安全
bcrypt/Argon2 慢哈希 + 登录后 session.regenerate() + 会话超时

### 6. CSP 安全策略
默认 self + 脚本白名单 + frame none + nonce 替 inline

### 7. 传输层安全
TLS 1.2+ + HSTS 强制 HTTPS + 安全响应头矩阵

### 8. 全生命周期安全
开发（验证+参数化）→ 部署（HTTPS+安全头）→ 运维（更新+监控+渗透）
