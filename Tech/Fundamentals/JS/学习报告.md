## 三、核心要点

1. **事件循环**是 JS 异步核心，微任务优先级 > 宏任务，Promise.then 是微任务，setTimeout 是宏任务
2. **this 绑定优先级**：new > 显式(bind/call/apply) > 隐式(obj.fn()) > 默认，隐式丢失是高频考点
3. **原型链**是 JS 继承机制，查找从自身 → __proto__ → 直到 null，delete 只删除自身属性
4. **闭包** = 函数 + [[Scope]]，不是内存泄漏，只有不合理持有大对象才会
5. **Promise.all** 一个 reject 全部失败；async/await 本质是 Generator 语法糖
6. **location.search** 含 `?` 前缀，**location.hash** 含 `#` 前缀
7. **防抖** = 停手后执行（重置计时器），**节流** = 固定间隔执行一次
8. **展开运算符**是浅拷贝且有额外性能开销，字面量对象创建更快
