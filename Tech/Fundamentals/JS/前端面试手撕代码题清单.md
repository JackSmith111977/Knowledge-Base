# 前端面试手撕代码题 · 完整题型清单

> 汇总前端面试中常考的手写代码题，按难度分组，标注考察知识点与推荐题源。

---

## JavaScript 基础手写题

### Easy

- **数组去重 `unique(arr)`**
  - 考察：Set、Map、filter + indexOf、哈希表
  - 题源：无特定 LeetCode 题，基础题

- **数组扁平化 `flatten(arr, depth?)`**
  - 考察：递归、Array.prototype.flat、concat + 展开运算符
  - 题源：LeetCode 不直接对应，属于基础手写

- **千分位格式化 `formatNumber(num)`**
  - 考察：字符串操作、正则表达式、`toLocaleString`
  - 题源：无特定 LeetCode 题

- **深拷贝 vs 浅拷贝 `deepClone(obj)` / `shallowClone(obj)`**
  - 考察：递归拷贝、循环引用（WeakMap）、类型判断（Object.prototype.toString）、Date/RegExp/Map/Set 等特殊对象处理
  - 题源：无特定 LeetCode 题

- **浅比较 `shallowEqual(obj1, obj2)`**
  - 考察：对象遍历、引用比较、React.memo 的 areEqual 逻辑
  - 题源：无特定 LeetCode 题

- **数组去重且排序 `uniqueAndSort(arr)`**
  - 考察：Set 去重 + sort 排序、时间复杂度分析
  - 题源：无特定 LeetCode 题

### Medium

- **防抖 `debounce(fn, delay, options?)`**
  - 考察：闭包、setTimeout/clearTimeout、this 指向、immediate 模式、取消/取回最新结果
  - 题源：无 LeetCode 对应，前端必考

- **节流 `throttle(fn, delay, options?)`**
  - 考察：时间戳 vs 定时器实现、leading/trailing 选项、this 和参数传递
  - 题源：无 LeetCode 对应，前端必考

- **`instanceof` 实现**
  - 考察：原型链遍历、`__proto__` / `Object.getPrototypeOf`、constructor.prototype
  - 题源：无 LeetCode 对应

- **`new` 操作符实现**
  - 考察：构造函数、原型链、`Object.create`、`apply` 绑定 this
  - 题源：无 LeetCode 对应

- **`call` / `apply` / `bind` 实现**
  - 考察：this 绑定、函数借用、上下文切换、参数展开、返回函数（bind 返回新函数）
  - 题源：无 LeetCode 对应

- **函数柯里化 `curry(fn)`**
  - 考察：闭包、递归、`fn.length` 判断参数个数、参数累积
  - 题源：无 LeetCode 对应

- **组合函数 `compose(...fns)` / `pipe(...fns)`**
  - 考察：函数式编程、reduce / reduceRight、函数组合
  - 题源：无 LeetCode 对应

- **数组原生方法实现**
  - `Array.prototype.map`、`Array.prototype.filter`、`Array.prototype.reduce`、`Array.prototype.forEach`
  - 考察：数组遍历、回调绑定、thisArg、边界条件（稀疏数组）
  - 题源：无 LeetCode 对应

- **`Object.is` 实现**
  - 考察：=== 的不足（NaN、+0 / -0）、类型判断
  - 题源：无 LeetCode 对应

- **发布订阅模式 `EventEmitter` / 事件总线**
  - 考察：事件注册/触发/解绑/一次性监听、回调队列、this 上下文
  - 题源：无 LeetCode 对应

- **模板字符串解析 `render(template, data)`**
  - 考察：正则匹配、字符串替换、嵌套数据访问
  - 题源：无 LeetCode 对应

- **JSONP 实现**
  - 考察：动态 script 标签创建、回调函数挂载与清理、Promise 封装
  - 题源：无 LeetCode 对应

- **简易路由实现（Hash / History 模式）**
  - 考察：`hashchange` / `popstate` 事件监听、路由表注册、页面刷新
  - 题源：无 LeetCode 对应

### Hard

- **Promise 核心实现**
  - `Promise` 构造函数（pending / fulfilled / rejected 三态转换）
  - `then` / `catch` / `finally` 链式调用
  - 考察：状态机、微任务（queueMicrotask / MutationObserver）、then 返回值穿透、链式 then 的 resolvePromise 逻辑
  - 题源：无 LeetCode 对应，但可参考 Promises/A+ 规范

- **`Promise.all` / `Promise.allSettled` / `Promise.race` / `Promise.any`**
  - 考察：并发控制、错误处理、状态聚合、计数器
  - 题源：无 LeetCode 对应

- **Promise 并发控制器（限制最大并发数）**
  - 考察：任务队列、并发调度、Promise 链、递归或迭代取下一个任务
  - 题源：类似 LeetCode 并发调度思路，无直接对应题

- **深拷贝完整实现（含循环引用、特殊类型）**
  - 考察：WeakMap 打破循环引用、Date/RegExp/Map/Set/ArrayBuffer/Symbol/Function 等特殊类型处理、原型链保留
  - 题源：无 LeetCode 对应

- **`deepEqual` 深度相等比较**
  - 考察：递归比较、类型区分、循环引用检测、Map/Set 遍历比较
  - 题源：无 LeetCode 对应

- **`Scheduler` 调度器实现**
  - 考察：任务优先级、并发限制、微任务调度、FIFO/LIFO 策略
  - 题源：无 LeetCode 对应

- **手写原生 AJAX / Fetch 封装**
  - 考察：XMLHttpRequest 状态机、Promise 封装、超时处理、取消请求（AbortController）
  - 题源：无 LeetCode 对应

- **`LRU` 缓存**
  - 考察：Map + 双向链表 / Map 的有序特性、get/put O(1) 时间复杂度
  - 题源：LeetCode 146. LRU Cache

## React 手写题

### Easy

- **`usePrevious(value)`**
  - 考察：useRef 保存上一次渲染值、生命周期差异
  - 题源：无 LeetCode 对应

- **`useMount(fn)` / `useUnmount(fn)`**
  - 考察：useEffect 空依赖数组、清理函数
  - 题源：无 LeetCode 对应

- **`useTitle(title)`**
  - 考察：useEffect 副作用、document.title 操作
  - 题源：无 LeetCode 对应

- **`useClickOutside(ref, handler)`**
  - 考察：事件监听与清理、document 级别事件、ref 引用
  - 题源：无 LeetCode 对应

### Medium

- **`useDebounce(value, delay)`**
  - 考察：useState + useEffect / setTimeout 结合、值同步、清除定时器
  - 题源：无 LeetCode 对应

- **`useThrottle(value, delay)`**
  - 考察：时间戳 / 定时器节流策略、useState + useEffect
  - 题源：无 LeetCode 对应

- **`useMemo` 简化实现**
  - 考察：Hook 链表、依赖数组 shallowEqual 比较、缓存值存储与复用
  - 题源：无 LeetCode 对应，重点理解 React 内部 Fiber 上的 memoizedState

- **`useCallback` 简化实现**
  - 考察：与 useMemo 的异同（useCallback 缓存函数、useMemo 缓存值）、依赖比较
  - 题源：无 LeetCode 对应

- **`useState` 简化实现**
  - 考察：闭包或全局 Hook 链表存储状态、setter 触发重新渲染、初始值仅在首次有效
  - 题源：无 LeetCode 对应

- **`useEffect` 简化实现**
  - 考察：副作用注册与执行时机、依赖变化检测、清理函数调用顺序
  - 题源：无 LeetCode 对应

- **`React.memo` 简化实现**
  - 考察：高阶组件（HOC）、props 浅比较、自定义 areEqual 函数
  - 题源：无 LeetCode 对应

- **`useImperativeHandle` + `forwardRef` 组合**
  - 考察：ref 转发、暴露子组件方法、命令式 API 设计
  - 题源：无 LeetCode 对应

- **`useReducer` 简化实现**
  - 考察：reducer 模式、dispatch 函数、与 useState 的关系
  - 题源：无 LeetCode 对应

### Hard

- **Mini React 实现（简易 React 框架）**
  - 考察：createElement、render 函数、Fiber 节点结构、工作循环（requestIdleCallback）、提交阶段（commit）、协调（reconciliation）
  - 题源：参考 build-your-own-react 项目，无 LeetCode 对应

- **Diff 算法简化实现**
  - 考察：同层比较、key 的作用、三种情况（节点新增/删除/移动）、最长递增子序列优化（React 16+）
  - 题源：无 LeetCode 对应，但 LIS 可参考算法题

- **Hooks 链表机制实现**
  - 考察：Fiber.memoizedState、Hook 链表顺序、为什么 Hook 不能在条件/循环中调用、多 Hook 类型的统一存储
  - 题源：无 LeetCode 对应

- **简易 Redux / 状态管理实现**
  - 考察：createStore（state、dispatch、subscribe、getState）、reducer 纯函数、中间件机制（compose）
  - 题源：无 LeetCode 对应

- **Context API 简化实现**
  - 考察：Provider/Consumer 模式、跨层级传值、 useContext Hook 原理
  - 题源：无 LeetCode 对应

## 算法类 / 场景类前端题

### Easy

- **图片懒加载**
  - 考察：IntersectionObserver API / scroll 事件 + getBoundingClientRect、data-src 替换、loading="lazy" 属性
  - 题源：无 LeetCode 对应

- **无限滚动（Infinite Scroll）**
  - 考察：IntersectionObserver 监听底部 sentinel、分页加载、loading 状态管理、防重复请求
  - 题源：无 LeetCode 对应

- **倒计时 / 定时器**
  - 考察：setInterval / requestAnimationFrame、时间计算、页面隐藏时暂停（visibilitychange）
  - 题源：无 LeetCode 对应

- **复制文本到剪贴板**
  - 考察：navigator.clipboard API、document.execCommand('copy') 降级方案、用户手势触发限制
  - 题源：无 LeetCode 对应

### Medium

- **虚拟列表（Virtual List）**
  - 考察：可视区域计算、滚动偏移量、startIndex / endIndex 计算、总高度占位、动态高度处理
  - 题源：无 LeetCode 对应，前端经典场景题

- **大文件分片上传**
  - 考察：File.slice() 分片、并发上传控制（Promise 并发调度）、断点续传（localStorage 记录进度）、MD5 文件校验、合并请求
  - 题源：无 LeetCode 对应

- **拖拽排序（Drag & Drop List）**
  - 考察：HTML5 Drag & Drop API / mouse 事件实现、元素位置计算、数组重排、动画过渡
  - 题源：无 LeetCode 对应

- **瀑布流布局**
  - 考察：多列高度追踪、最短列优先放置、resize 响应、图片加载完成后再布局
  - 题源：无 LeetCode 对应

- **搜索框自动补全（Autocomplete / Typeahead）**
  - 考察：防抖、异步请求竞态处理（AbortController）、键盘导航（上下键选择）、高亮匹配文本
  - 题源：无 LeetCode 对应

- **表格列宽拖拽调整**
  - 考察：mouse 事件监听、列宽计算、边界限制、性能优化（requestAnimationFrame）
  - 题源：无 LeetCode 对应

- **进度条 / 上传进度**
  - 考察：XMLHttpRequest.upload.onprogress、百分比计算、动画平滑过渡
  - 题源：无 LeetCode 对应

### Hard

- **在线代码编辑器（简易版 CodePen）**
  - 考察：iframe 沙箱隔离、postMessage 通信、实时预览、错误捕获、代码格式化
  - 题源：无 LeetCode 对应

- **简易 Markdown 解析器**
  - 考察：正则匹配、AST 构建、HTML 渲染、代码高亮、嵌套语法处理
  - 题源：无 LeetCode 对应

- **前端错误监控 SDK**
  - 考察：window.onerror、unhandledrejection 监听、Vue/React 错误边界、资源加载错误（addEventListener('error', true)）、堆栈解析、上报策略（ Beacon API、批处理）
  - 题源：无 LeetCode 对应

- **时间线 / Gantt 图渲染**
  - 考察：时间轴计算、任务重叠处理、动态缩放、滚动同步
  - 题源：无 LeetCode 对应

- **可撤销/重做系统（Undo/Redo）**
  - 考察：命令模式、操作栈管理、状态快照 vs 增量 diff、内存优化
  - 题源：无 LeetCode 对应
