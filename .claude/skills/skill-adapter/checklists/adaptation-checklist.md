## 常见问题

### Q1: 脚本路径找不到
**解决：** 确认 `.claude/scripts/` 目录是否存在，脚本文件是否已复制

### Q2: Hook 不触发
**解决：** 检查 `settings.local.json` 配置是否正确，命令路径是否可访问

### Q3: 适配项遗漏
**解决：** 重新运行扫描，或手动检查 SKILL.md 中的路径引用

### Q4: 触发词与新项目冲突
**解决：** 检查新项目已有 Skill 的触发词，调整迁移 Skill 的 `triggers` 配置避免冲突

### Q5: 命令名称重复
**解决：** 修改 `commands` 字段，使用唯一的命令名称（如 `/skill-name-v2`）

### Q6: Hooks 脚本路径错误（%CD% 未展开）
**解决：** 检查 hooks command 中的环境变量 `%CD%` 是否被正确展开，建议改用绝对路径

### Q7: Hooks 脚本存在但执行失败
**解决：** 检查路径分隔符（Windows 使用 `\\` 双反斜杠）、空格处理（路径加引号）
