# study - 费曼学习法互动教学 Skill

## 用途

基于**费曼学习法**的互动教学 Skill，提供两种学习模式：

| 模式 | 流程 | 适用场景 | 预计耗时 |
|------|------|----------|----------|
| **标准模式** | 7 步：概念→讲解→查漏→重构→逐题测验→复述→评分 | 深度学习新概念 | 30-40 分钟 |
| **快速模式** | 5 步：概念→讲解→逐题测验→评分 | 复习/快速浏览 | 15-20 分钟 |

**核心原则：** 以教促学、逐题追问、评分反馈、小节覆盖

---

## 工作流程

### 阶段 0：启动与模式选择

**步骤 0.1：欢迎与模式选择**

```markdown
📚 **费曼学习法学习向导**

请选择学习模式：

**🐢 标准模式**（约 30-40 分钟）
- 7 步流程：概念→讲解→查漏→重构→逐题测验→复述→评分
- 包含比喻、深度复述、详细评分
- 适合：首次学习复杂概念

**🐇 快速模式**（约 15-20 分钟）
- 5 步流程：概念→讲解→逐题测验→评分
- 精简流程，聚焦核心测验
- 适合：复习或简单概念

请回复「标准」或「快速」选择模式。
```

**步骤 0.2：主题确认**

- **方式 A**：用户指定主题
- **方式 B**：扫描知识库推荐

**步骤 0.3：知识库内容加载**

加载成功后，显示章节结构，用户确认开始。

### 阶段 1-3：学习流程（分模式）

**标准模式：** 详见 `references/standard-mode.md`
**快速模式：** 详见 `references/quick-mode.md`

**核心机制：**
| 机制 | 说明 |
|------|------|
| **逐题追问** | 每次只问 1 题，根据回答决定下一题 |
| **评分反馈** | 每题 0-5 分，附点评和纠正 |
| **小节覆盖** | 按知识库小节结构逐个击破 |
| **章节追踪** | 记录已学章节，完成后询问是否继续 |

**章节切换逻辑：**

```markdown
## 章节完成检查点

完成一个章节后（Step 5 复习计划后），执行以下检查：

1. **显示当前进度**
   - 已学章节：[章节列表]
   - 剩余章节：[章节列表]

2. **询问用户**
   - 回复「继续」→ 进入下一章节
   - 回复「结束」→ 生成总学习报告
   - 回复「跳过 [章节名]」→ 跳过指定章节

3. **如选择继续**
   - 加载下一章节内容
   - 从 Step 1 重新开始流程
   - 记录章节进度
```

**多章节学习完成后的总总结：**

```markdown
## 总学习总结

完成所有章节后，生成综合报告：
- 各章节评分汇总
- 整体掌握度评估
- 综合复习计划
```

### 阶段 5：章节切换（新增）

**在每个章节完成后执行：**

```markdown
**步骤 5.1：显示章节进度**

章节完成时显示：
- ✅ 已学章节：[列表]
- ⏳ 剩余章节：[列表]
- ➡️ 下一步操作？

**步骤 5.2：处理用户选择**

| 用户回复 | 动作 |
|----------|------|
| 「继续」 | 加载下一章，回到阶段 1 |
| 「跳过 X」 | 标记跳过，进入下一章 |
| 「结束」 | 生成总学习报告，进入阶段 6 |
| 「复习」 | 重新学习当前章节 |

**步骤 5.3：章节加载**

加载下一章内容前：
1. 保存当前章节进度
2. 更新 currentChapter
3. 从步骤 1 重新开始
```

## 评分标准

详见 `references/scoring-rubric.md`

| 分数 | 标准 |
|------|------|
| **5 分** | 准确完整，能用自己的话解释，无知识盲点 |
| **4 分** | 基本正确，有小遗漏或表述不够清晰 |
| **3 分** | 核心概念正确，有明显遗漏或误解 |
| **2 分** | 部分正确，但关键概念有错误 |
| **1 分** | 回答与问题关联度低 |
| **0 分** | 未回答或完全错误 |

---

## 踩坑清单

| 陷阱 | 错误做法 | 正确做法 |
|------|----------|----------|
| **术语堆砌** | 允许用户使用行话 | 强制用简单语言解释 |
| **跳过盲点** | 用户说不懂就直接给答案 | 引导用户自己发现并解决 |
| **一次性多问** | 一次抛出多个问题 | 逐题追问，根据回答调整 |
| **不评分** | 只给点评 | 每题必须打分，记录追踪 |
| **不复盘** | 学完就结束 | 必须生成总结和复习计划 |
| **章节切换缺失** | 完成一章直接结束 | 每章完成后询问是否继续 |
| **进度不追踪** | 不记录已学章节 | 维护 learnedChapters 列表 |
| **超纲出题** | 出后续章节的内容 | 题目严格限制在当前 Step 1 已学范围内 |
