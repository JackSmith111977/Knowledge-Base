## 使用说明

### 查询推荐位置

1. **主题已收录** → 直接使用"推荐目录"列的路径
2. **主题未收录** → 分析主题属性后推荐：
   - 前端框架/库 → `Tech/Frameworks/[框架名]/`
   - 后端框架/数据库 → `Tech/Backend/[框架名]/`
   - AI/ML 相关 → `Tech/AI/[技术名]/`
   - 构建工具 → `Tech/BuildTools/[工具名]/`
   - 业务组件/方案 → `Tech/Business/[方案名]/`
   - 基础知识 → `Tech/Fundamentals/[分类]/`

### 更新索引

当创建新文档到未收录的目录时，自动更新此索引：
1. 添加新主题到"主题分类映射"表格
2. 更新"最后更新"日期
3. 如有新目录，更新"目录结构"
