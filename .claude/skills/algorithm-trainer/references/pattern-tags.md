# 模式标签体系（Pattern Tag System）

> "什么特征 → 什么解法" 的决策树记录
> 这是算法学习的核心——训练模式识别能力，而不是记忆代码

## 为什么需要模式标签

看到题目下意识想用暴力？说明缺乏"特征 → 方法"的映射。
模式标签就是你要练的肌肉记忆：

```
题目描述特征 → 识别算法类型 → 选择具体模板
```

## 标签格式

```
[算法大类] 具体模式：特征描述 → 解法框架 → 关键公式
```

## 滑动窗口专题

### [滑动窗口] 定长窗口
- **特征**：求长度为 k 的子数组/子串的某种统计值（最大/最小/平均值）
- **解法**：维护一个大小为 k 的窗口，每次右移一步
- **关键**：窗口滑动时只需更新两个元素（新进和出去），O(n)
- **典型题**：643（最大平均数）、1456（元音最大数）

### [滑动窗口] 不定长 - 求最长
- **特征**：求满足某条件的最长子数组/子串，"越长越不合法"
- **解法**：右指针扩展，不合法时左指针收缩
- **关键公式**：`ans = max(ans, right - left + 1)`
- **典型题**：3（无重复最长子串）、1004（最大连续1的个数III）

### [滑动窗口] 不定长 - 求最短
- **特征**：求满足某条件的最短子数组/子串，"越长越合法"
- **解法**：右指针扩展，合法时尝试左指针收缩
- **关键公式**：`ans = min(ans, right - left + 1)`
- **典型题**：209（长度最小子数组）、76（最小覆盖子串）

### [滑动窗口] 求子数组个数
- **特征**：求满足条件的子数组/子串数量
- **解法**：越短越合法时 `ans += right - left + 1`，越长越合法时 `ans += left`
- **关键**：内层循环只移动一个指针
- **典型题**：713（乘积小于K）、1358（包含三种字符）

### [滑动窗口] 恰好型
- **特征**：恰好等于 k / 恰好 k 个不同
- **解法**：转化为两个"至少"之差：`atMost(k) - atMost(k-1)`
- **典型题**：992（K个不同整数）、930（和相同的二元子数组）

### [双指针] 相向双指针
- **特征**：有序数组中找两个/三个/四个数满足条件
- **解法**：left=0, right=n-1，根据大小关系移动
- **典型题**：15（三数之和）、11（盛最多水）

### [双指针] 同向双指针
- **特征**：在原数组上标记/覆盖，需要保持某些顺序
- **解法**：快指针遍历，慢指针标记写入位置
- **典型题**：283（移动零）、26（删除重复项）

### [分组循环]
- **特征**：数组可以分成若干段，每段内部有共同特征
- **解法**：外层循环标记段起点，内层循环找到段终点
- **典型题**：485（最大连续1）、413（等差数列划分）

## 二分算法专题

### [二分查找] 红蓝染色法
- **特征**：有序数组中找满足条件的第一个位置
- **解法**：定义 check(mid)，左侧蓝色（合法）右侧红色（不合法）
- **关键公式**：`lowerBound(nums, x)` 返回 ≥x 的第一个位置
- **典型题**：35（搜索插入位置）、34（查找首末位置）

### [二分答案] 求最小值
- **特征**：最小化某个值，check(mid) 可以 O(n) 验证
- **解法**：check(mid)==true → right=mid，反之 left=mid
- **典型题**：875（爱吃香蕉）、1011（D天送达）

### [二分答案] 求最大值
- **特征**：最大化某个值，check(mid) 可以 O(n) 验证
- **解法**：check(mid)==true → left=mid，反之 right=mid
- **典型题**：1642（最远建筑）、2226（分糖果）

### [二分答案] 最小化最大值
- **特征**：将数组分组，最小化每组的最大值
- **解法**：二分这个"最大值"，验证能否分成满足条件的组
- **典型题**：410（分割数组）、1631（最小体力消耗）

## 数据结构专题

### [前缀和] 一维
- **特征**：频繁查询子数组和/异或和
- **解法**：`sum[left..right] = prefix[right+1] - prefix[left]`
- **典型题**：303（区域和检索）、1310（异或查询）

### [前缀和] + 哈希表
- **特征**：求和为 k 的子数组个数/最长子数组
- **解法**：枚举右端点，用哈希表记录前缀和出现次数
- **关键公式**：`count[prefix - k]` 即为满足条件的左端点个数
- **典型题**：560（和为K的子数组）、525（连续数组）

### [差分]
- **特征**：区间修改 + 单点查询 / 区间修改后查询最终状态
- **解法**：在区间起点 +val，区间终点后 -val，最后做前缀和
- **典型题**：1109（航班预订）、1854（人口最多）

### [单调栈]
- **特征**：找左边/右边第一个比当前元素大/小的元素
- **解法**：维护单调递增/递减栈
- **典型题**：739（每日温度）、84（柱状图最大矩形）

### [单调队列]
- **特征**：滑动窗口求最值
- **解法**：维护窗口内的单调递增/递减队列
- **典型题**：239（滑动窗口最大值）

### [堆] Top-K
- **特征**：数据流中第 K 大/小、前 K 个最大/最小
- **解法**：维护大小为 K 的最小/最大堆
- **典型题**：703（数据流第K大）、347（前K个高频）

### [堆] 多路归并
- **特征**：合并多个有序序列
- **解法**：每个序列头部入堆，弹出后推进
- **典型题**：23（合并K个链表）、632（最小区间）
