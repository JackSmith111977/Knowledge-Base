# 间隔复习机制（Spaced Repetition）

> 3-5-7 法则：根据首次掌握程度决定复习间隔

## 核心原理

根据艾宾浩斯遗忘曲线，新学知识在 2-3 天后遗忘速度最快。此时刻意回忆效果最好。

## 间隔规则

### 首次通过（评分 5 分）

| 轮次 | 间隔 | 说明 |
|------|------|------|
| 首次复习 | 3 天后 | 刚学完 3 天，记忆开始衰退 |
| 二次复习 | 5 天后 | 距首次复习 5 天 |
| 三次复习 | 7 天后 | 距二次复习 7 天，之后标记为"已掌握" |

### 有提示通过（评分 3 分）

| 轮次 | 间隔 | 说明 |
|------|------|------|
| 首次复习 | 2 天后 | 需要加强 |
| 二次复习 | 3 天后 | |
| 三次复习 | 5 天后 | |

### 看题解通过（评分 1 分）

| 轮次 | 间隔 | 说明 |
|------|------|------|
| 首次复习 | 1 天后 | 必须次日就复习 |
| 二次复习 | 2 天后 | |
| 三次复习 | 3 天后 | |

## 复习队列算法

```
每日加载进度时：
1. 遍历 completedProblems
2. 对每道题，检查 nextReviewDate
3. 如果 nextReviewDate <= 今日日期 → 加入今日复习队列
4. 如果复习队列 > 5 道 → 优先保留错题和间隔短的
```

## 题目状态流转

```
新题 → 完成 → reviewCount=0, masteryLevel=首次评分
  ↓
复习轮次 1：评分变化 → masteryLevel 更新
  ↓ 如果评分提升 → 间隔增大
  ↓ 如果评分下降 → 间隔缩短，加入错题本
  ↓
复习轮次 2：...
  ↓
复习轮次 3：如果 ≥ 3 分 → 标记 mastered=true
  ↓ 如果 < 3 分 → 重置 reviewCount=0，重新开始间隔
```

## 错题优先级

复习队列中题目排序优先级：
1. 错题（score ≤ 3）优先
2. 间隔天数少的优先
3. 同间隔下，难度分高的优先

## 特殊规则

- **连续 3 次复习满分**：跳过后续复习，标记 mastered
- **复习时再次做错**：重置为"新题"状态，从 1 天间隔重新开始
- **超过 14 天未复习**：间隔不继续增长，保持在 7 天，但标记为 overdue
